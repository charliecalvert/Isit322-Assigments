Testing Require.js code with Karma
==================================

# Run with one of the following:

    karma start
    grunt test
    grunt karma

## Turn it in

Get all the tests to pass. Make sure jshint comes back clean.

Place your work in your repository in a folder called Week02_FunctionBasics.
